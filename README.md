
# techmahotsav

## Run Locally

Go to the project directory

```bash
  cd techmahotsav
```

Install dependencies

```bash
  npm install
```

Rename `.env.example` to `.env` and add these fields

```bash
  PORT=
  MONGO_URI=
  JWT_SECRET=
```

Testing

```bash
  npm test
```

Start the server

```bash
  npm run start
```
